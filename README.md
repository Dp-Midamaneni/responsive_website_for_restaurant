# Food Munch Website

A responsive food-themed static website built using **HTML**, **CSS**, **Bootstrap 4**, and **Font Awesome**.  
This project presents a modern restaurant landing page with sections like **Why Choose Us**, **Explore Menu**, **Delivery & Payment**, **Thanking Customers**, and **Follow Us**.

## Features

- Responsive navigation bar
- Attractive hero banner
- Why Choose Us section with service highlights
- Explore Menu section with food category cards
- Healthy food highlight section
- Delivery and Payment section
- Thanking Customers section
- Social media follow section
- Footer with contact information

## Technologies Used

- HTML5
- CSS3
- Bootstrap 4
- Font Awesome
- Google Fonts
